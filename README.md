# 广金逼乎
广金大学生吐槽的地方

## 环境要求

1. nodejs > v6.9
    - 推荐使用[nvm](https://github.com/coreybutler/nvm-windows)管理node版本
2. npm 随node安装

3. 小程序开发工具

4. 编辑器vscode, webstorm

## 生成新页面

```
gulp auto -s index -p mypage

复制pages/index中的文件创建名称为mypage的页面
```

## 开发流程

```
npm install

npm run dev
```

## 测试

```
npm run test
```

## 编译流程

```
npm run build
```
